A. BABAEI PORTFOLIO V4 — EXACT UPLOAD PROCEDURE

1. Extract the ZIP.
2. Open the GitHub repository A-Babaei/A-Babaei.github.io.
3. Delete old website files or replace them with the files listed below.
4. Upload every listed root file directly to the repository root.
5. Upload the complete media folder without renaming it.
6. Do not upload the outer a-babaei-portfolio-v4 folder.
7. Commit to main.
8. Settings > Pages > Deploy from a branch > main > /(root).
