Repository review

What must be corrected before promotion.

01

Rename clearly

Use a professional repository name such as mathematical-function-animations.

02

Real source files

Move code from README examples into actual .m files and keep the Live Script as an optional demonstration.

03

License

Add an actual LICENSE file with your correct name and year rather than placeholder text.

04

Visual previews

Add compressed GIF or WebM previews so each animation is understandable before MATLAB is opened.

05

Requirements

State tested MATLAB versions, toolbox requirements, performance expectations, and how to stop each animation.

06

Documentation

Explain the mathematical concept, parameters, expected output, and known numerical singularities.
