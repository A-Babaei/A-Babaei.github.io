# A. Babaei Portfolio V5

A static, dependency-free GitHub Pages portfolio optimized for desktop and mobile.

## Performance

- Native scroll-snap galleries
- Frame-limited Canvas animation
- Offscreen animation pausing
- Reduced particle density on mobile
- No frameworks, remote fonts, or animation libraries
- Mobile backdrop filters disabled

## Edit content

Edit `site-data.js` and replace preview files inside `media/`.

## Profile image

Upload `profile-photo.jpg` to the repository root. Recommended: 1200 × 1500 px, under 500 KB.

## Public-content protection

Visible web content cannot be made impossible to copy. This project uses watermarked previews and media drag/context-menu deterrence. Do not upload valuable source files to a public repository.
