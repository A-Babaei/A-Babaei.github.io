# A. Babaei Portfolio V4

Minimal dark editorial portfolio for GitHub Pages with glass surfaces, evidence sliders, website case studies, protected preview deterrents, and a procedural neural silhouette animation.

Edit content in `site-data.js`. Replace SVG placeholders inside `media/` with optimized JPG/WebP previews using the same filenames, or update the image paths in `site-data.js`.

Read `UPLOAD_INSTRUCTIONS.txt` and `PROTECTION_NOTES.txt` before publishing.
